# Kronos Shield

Kronos Shield is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** and **Windows** systems. 
It protects user privacy by verifying VPN status and blocking sensitive add-ons when no secure tunnel is detected.

---

## Features
- **VPN Detection** – Confirms if a VPN tunnel (e.g., tun0) is active. 
- **Startup Check** – Automatic verification at boot with smart delays for stability. 
- **Background Watchdog** – Monitors connection status silently during use. 
- **Add-on Protection** – Blocks video add-ons such as streaming or P2P services when no VPN is found. 
- **User Tools** – Includes a simple GUI with a “Check VPN Status” button for manual inspection. 
- **Fail-Safe Design** – Lightweight, non-intrusive, and always abort-safe.

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Windows 10/11 
- Python 3 (bundled with Kodi)
